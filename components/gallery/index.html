Cada sección muestra una funcionalidad del componente con un ejemplo en vivo y el código necesario.

Import

import './gallery.js';

Basic

La galería necesita un array de imágenes y las etiquetas <gallery-component>. Cada imagen tiene una URL grande (src) y una miniatura (thumb).

<!-- HTML --> <gallery-component id="gallery"></gallery-component> <!-- JavaScript --> import './gallery.js'; const gallery = document.getElementById('gallery'); gallery.images = [ { src: 'foto.jpg', thumb: 'thumb.jpg', alt: 'Descripción' }, ... ];

Thumbnails

Las miniaturas se muestran debajo de la imagen principal. Se pueden ocultar con el atributo show-thumbnails="false".

<!-- Sin miniaturas --> <gallery-component show-thumbnails="false"> </gallery-component> <!-- Con miniaturas (por defecto) --> <gallery-component></gallery-component>

Caption

Muestra un texto descriptivo sobre la imagen. Se activa con show-caption="true". Los datos se pasan en title y desc de cada imagen.

<!-- Con caption --> <gallery-component show-caption="true"> </gallery-component> // Datos de ejemplo { src: '...', thumb: '...', alt: 'Montañas', title: 'Montañas', desc: 'Vista panorámica' }

Fullscreen

La galería se puede poner a pantalla completa. Se controla con la propiedad fullscreen. También funciona con doble clic en la imagen o con la tecla F.

// JavaScript gallery.fullscreen = true; // Abrir gallery.fullscreen = false; // Cerrar // Eventos gallery.addEventListener('fullscreen-toggle', () => { // Se dispara con doble clic en la imagen });

Responsive

El componente tiene 3 tamaños predefinidos que se controlan con el atributo viewport.

ValorAnchoAlto imagen
large1000px440px
medium803px342px
small640px274px
<!-- Tamaños --> <gallery-component viewport="large"></gallery-component> <gallery-component viewport="medium"></gallery-component> <gallery-component viewport="small"></gallery-component> // JavaScript gallery.viewport = 'large';

API — CSS Parts

Para personalizar los estilos desde fuera del Shadow DOM usa el pseudoelemento ::part().

/* Personalizar desde tu CSS */ gallery-component::part(container) { border-radius: 8px; } gallery-component::part(image) { object-fit: contain; } gallery-component::part(thumbnails) { background: #fff; } gallery-component::part(caption) { color: #fff; }
PartElemento
containerLa galería entera
slideÁrea de la imagen principal
imageLa imagen grande
captionEl texto sobre la imagen
thumbnailsLa tira de miniaturas
prev-btnBotón anterior
next-btnBotón siguiente
