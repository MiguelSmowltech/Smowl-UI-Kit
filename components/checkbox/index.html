1. Normal

El checkbox sin marcar. Es el estado inicial. El texto se pone dentro de la etiqueta.

Acepto los términos y condiciones 
<!-- Sin marcar. Así se escribe en HTML: -->
<checkbox-component>Acepto los términos y condiciones</checkbox-component>

2. Marcado

El checkbox empieza marcado. Solo hay que añadir el atributo checked.

Recibir notificaciones 
<!-- Marcado desde el principio -->
<checkbox-component checked>Recibir notificaciones</checkbox-component>

3. Deshabilitado

El checkbox no se puede usar. El atributo disabled lo bloquea. Sirve para opciones que no están disponibles.

Opción bloqueada Opción bloqueada (marcada) 
<!-- Deshabilitado (no se puede hacer clic) -->
<checkbox-component disabled>Opción bloqueada</checkbox-component>
<checkbox-component checked disabled>Opción bloqueada (marcada)</checkbox-component>

4. Con valor (value)

El atributo value guarda un texto que identifica esta opción. Sirve para formularios.

Suscribirse al newsletter value="newsletter" 
<!-- Con un valor para formularios -->
<checkbox-component value="newsletter" checked>
  Suscribirse al newsletter
</checkbox-component>

5. Control desde JavaScript

Puedes marcar o desmarcar el checkbox usando código JavaScript. Pulsa el botón "Cambiar" para probarlo.

Modo oscuro 
// Cambiar el estado desde JavaScript:
const cb = document.getElementById('controlled');
cb.checked = true;   // Marcar
cb.checked = false;  // Desmarcar

6. Escuchar cambios

Puedes escuchar el evento "change" para saber cuándo alguien marca o desmarca. El menú lateral muestra el último evento.

Haz clic en mí 
// Escuchar cuándo cambia el estado:
componente.addEventListener('change', (e) => {
  console.log(e.detail.checked);
  // e.detail.checked → true o false
});

Resumen de la API

Todo lo que puedes hacer con este componente.

Nombre Tipo ¿Qué hace?
checked Atributo HTML o propiedad de JavaScript Marca o desmarca el checkbox. true = marcado, false = desmarcado.
disabled Atributo HTML o propiedad de JavaScript Deshabilita el componente. No se puede hacer clic ni con el teclado.
value Atributo HTML Guarda un texto que identifica esta opción. Por defecto vale "on".
change Evento de JavaScript Se dispara cuando el usuario marca o desmarca. El estado se lee con e.detail.checked.
