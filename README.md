# Smowl UI Kit

Interactive component demos and documentation for the Smowltech Design System.

## Structure

```
components/
  gallery/     — PrimeVue Galleria component demo with controls and docs
```

## Usage

Each component has its own directory with an `index.html` that serves as an interactive demo. Open any `index.html` in a browser to test the component live.

## Components

| Component | Description | Status |
|-----------|-------------|--------|
| [Gallery](components/gallery/) | Image gallery with thumbnails, fullscreen, keyboard nav | Active |
